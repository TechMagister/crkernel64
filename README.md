# crkernel64

64-bits kernel that can launch crystal code.

## Usage

Available targets for make :
- kernel : create kernel-x86_64.bin in the build folder
- iso : create a bootable iso ( build/os-x86_64.iso )
- run : Run QEMU ( qemu-system-x86_64 )
